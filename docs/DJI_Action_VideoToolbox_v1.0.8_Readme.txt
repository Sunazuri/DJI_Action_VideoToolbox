DJI_Action_VideoToolbox_v1.0.8 取扱説明書

概要
DJI Actionシリーズ動画のD-Log M→Rec.709変換、HEVC/NVENCエンコード、動画連結に対応したWindows用GUIツールです。

v1.0.8の主な変更点
・エンコード中でも「処理完了後シャットダウン」のON/OFFを変更可能にしました。
・エンコード中でも「エラー終了時も強制シャットダウン」のON/OFFを変更可能にしました。
・エンコード中でも待機時間と「秒 / 分」の単位を変更可能にしました。
・シャットダウン判定は、エンコード開始時ではなく完了直後の最新設定を採用します。
・キャンセル終了時はシャットダウンしません。
・エラー終了時は通常シャットダウンしません。
・「エラー終了時も強制シャットダウン」がONの場合のみ、詳細ログ保存後に指定時間経過でシャットダウンします。
・シャットダウン待機中はキャンセル可能です。

基本手順
1. ffmpeg.exe と ffprobe.exe を指定します。
2. エンコードタブを開きます。
3. LUTファイル、出力先、入力動画を指定します。
4. 処理モードとLUT補間方式を選びます。
5. 必要に応じて「処理完了後シャットダウン」をONにします。
6. 待機時間を数字入力欄へ入力し、単位を「秒」または「分」から選択します。
7. エラー終了時もシャットダウンしたい場合のみ「エラー終了時も強制シャットダウン」をONにします。
8. エンコード開始を押します。

シャットダウン機能の注意
・デフォルトはOFFです。
・OFF時は待機時間入力欄と単位プルダウンが無効化されます。
・ON時のみ待機時間入力欄と単位プルダウンが有効化されます。
・エンコード中にON/OFFを変更した場合、完了直後の最新状態が採用されます。
・誤操作防止のため、エラー終了時も強制シャットダウンは必要な場合のみONにしてください。

NVIDIA以外のGPUについて
本ツールのHEVC高速エンコードは主にNVIDIA NVENCを想定しています。AMD RadeonやIntel内蔵GPUのみの環境では、hevc_nvencを使う処理が失敗する可能性があります。その場合は、CPUエンコード、hevc_amf、hevc_qsvなどに対応した別設定が必要です。
